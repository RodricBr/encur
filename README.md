</br>

## - Instalação:

> git clone https://github.com/rodricbr/encur.git </br>
> cd encur/;chmod +x encur </br>
## - Uso:
> bash encur </br>
> ./encur </br>
## - Opções:
```markdown
./encur -u | -url  :: Executar o encur
./encur            :: Mostrar o painel de ajuda
```
